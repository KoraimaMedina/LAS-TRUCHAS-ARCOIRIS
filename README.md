# LAS-TRUCHAS-ARCOIRIS
Infografía web que  presentan visualmente cómo la dieta, la frecuencia de alimentación y el ambiente impactan el rendimiento productivo y económico de la trucha arcoíris. El diseño sigue un enfoque científico moderno con revelado progresivo de datos: las cifras y gráficos permanecen ocultos hasta que el usuario interactúa.
